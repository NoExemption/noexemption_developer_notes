## 👋 简介 (Introduction)

欢迎来到 **noexemption_developer_notes**。

这是一个个人的数字花园，专门用于存放我的备忘录、`.md` 文档以及个人相关的学习笔记。在这个日新月异的技术世界里，这里记录着我探索真理的每一个脚印。

## 📂 仓库内容 (Contents)

本仓库主要包含以下几类内容：

*   📝 **个人备忘录 (Memos):** 开发过程中遇到的 "坑" 与解决方案速查。
*   📚 **学习笔记 (Study Notes):** 计算机科学基础、编程语言、框架与工具的系统性学习记录。
*   💡 **技术随笔 (Thoughts):** 关于代码质量、架构设计或职业发展的碎片化思考。
*   🔧 **最佳实践 (Best Practices):** 收集的代码片段与配置清单。

## 🚀 目标 (Goals)

*   **沉淀知识**：将隐性知识转化为显性文档。
*   **持续精进**：保持 "进一寸有一寸的欢喜" 的心态，不断扩展技术边界。
*   **方便检索**：建立个人的知识索引，提高开发效率。
